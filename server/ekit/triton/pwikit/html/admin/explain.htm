MAP Pre-Workshop Interview Kit - Admin Console.

The menus on the left provide a set of administrative tools to allow you to

- Add a new workshop participant

- Add a boss to an existing workshop participant

- Add a peer to an existing workshop participant

- List all participants in the database

- Find a participant by id, name or boss name



If you are showing a list of participants, you are able to look at the individual's status, and even view the responses to the forms. You can see the necessary passwords and re-send emails if necessary.

The system is designed to be simple to use, and we hope that you have fun using it !
